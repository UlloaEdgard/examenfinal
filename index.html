Data Project

En nuestra página web, ofrecemos cursos de programación accesibles a todos los públicos,
desde S/. 1.00. Nuestro objetivo es introducir al mundo de la programación a personas con pocos recursos o que quieran potenciar sus conocimientos para trabajar en empresas o de forma independiente.

Mayor información Visítanos

HTML

Lenguaje para crear páginas web. Se debe estudiar porque es el lenguaje básico para crear páginas web.
Duración: 3 Meses

CSS

Lenguaje para dar formato a las páginas web. Se debe estudiar porque es esencial para crear páginas web atractivas.
Duración: 3 Meses

JavaScript

Lenguaje para agregar interactividad a las páginas web. Se debe estudiar porque es esencial para crear páginas web modernas.
Duración: 3 Meses

Python

Lenguaje versátil y potente que tiene muchas aplicaciones potenciales. Se debe estudiar porque es una herramienta valiosa para cualquier persona que quiera aprender a programar.
Duración: 3 Meses

PhP

Lenguaje de programación web popular, sencillo y potente que debes aprender. Se utiliza para crear páginas web dinámicas y aplicaciones web.
Duración: 3 Meses

SQL

Lenguaje esencial para trabajar con bases de datos. Debes prender SQL porque te abrirá muchas puertas profesionales, ya que es una habilidad muy demandada en el mercado laboral.
Duración: 3 Meses

C

Lenguaje potente que te permitirá crear aplicaciones y sistemas operativos eficientes.
Duración: 3 Meses

C++

Lenguaje de programación que te beneficiará a largo plazo, tanto en términos de conocimientos como de oportunidades profesionales.
Duración: 3 Meses

Visual Basic

Lenguaje de programación de propósito general que se utiliza para crear aplicaciones para Windows.
Duración: 3 Meses
